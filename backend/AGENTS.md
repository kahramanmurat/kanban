# Backend Overview

This directory contains the Python FastAPI backend for the Project Management MVP.

## Current scope

- Serves the exported frontend assets at `/`.
- Enforces a simple session-cookie login flow for the MVP credentials.
- Exposes a simple JSON health endpoint at `/api/health`.
- Exposes login and logout routes at `/api/login` and `/api/logout`.
- Exposes persistent board routes at `/api/board`, `/api/columns/...`, and `/api/cards/...`.
- Exposes a narrow AI connectivity route at `/api/ai/connectivity` for authenticated verification of the OpenAI setup.
- Exposes an authenticated AI board route at `/api/ai/board` that returns an assistant message plus an updated board snapshot.
- Creates and seeds a SQLite database automatically at `backend/data/pm.sqlite3`.
- Uses `uv` for dependency management and locking.
- Includes a focused pytest suite for auth and scaffolded routes.

## Structure

- `pyproject.toml`: Python project metadata and dependencies.
- `uv.lock`: Locked Python dependencies generated by `uv`.
- `app/db.py`: SQLite schema, seed data, and board persistence helpers.
- `app/main.py`: FastAPI application entry point.
- `app/static/`: Static assets served by the backend. In Docker builds, this directory is replaced with the exported frontend output.
- `tests/test_app.py`: Backend tests for auth and base routes.
