# Scripts Overview

This directory contains local helper scripts for starting and stopping the Docker-based app.

## Current scripts

- `start-mac.sh`
- `stop-mac.sh`
- `start-linux.sh`
- `stop-linux.sh`
- `start-windows.ps1`
- `stop-windows.ps1`

Each script resolves the project root and runs the matching `docker compose` command from there.
